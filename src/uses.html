This site

This site is made with 11ty as my static site generator of choice, with nunjucks as my templating language. This tutorial was immensely helpful for me in getting started with 11ty! I also use flexbox as my choice of layout module, which allows my site to be resized for other screen sizes! It's hosted on the superior free static website hosting service known as Nekoweb, and deployed using Deploy2Nekoweb.

I do all of my coding in VSCodium, which is a free, open source software binary distribution of Microsoft's VSCode that doesn't contain any telemetry/tracking!

Operating System

I use EndeavourOS as my choice of operating system for my desktop PC, which is an Arch-based Linux distribution! It's a very lightweight distro that ships with a minimum amount of preinstalled apps, making it almost like a blank canvas that's ripe for personalization. Highly recommend it if you're looking for the Arch Linux experience, but want something a bit easier to install and manage than Arch itself.

On my laptop, I currently use Linux Mint, which is an Ubuntu-based distro, and I'm running it using the Cinnamon desktop environment. I only use my laptop when I'm travelling, or too sick to use my desktop computer, so I don't use it a ton other than in those instances. I'm thinking of switching to Fedora on it though, as I way prefer KDE to Cinnamon at this point but want something that's still easy to use.

Art

For digital art, I use Procreate on an iPad 9th generation with a 1st generation Apple Pencil the most. I also use Krita on an XP-Pen Artist Pro 13.3", but only on rare occasions since this tablet causes me a great amount of pain.

For 3D modeling, I use Nomad Sculpt on my iPad for the actual sculpting, CozyBlanket for retopology, and B For Artists for any final adjustments.

Traditionally, I use Ohuhu alcohol markers, Prismacolor Scholar pencil crayons, and Gellyroll gel pens on Ohuhu 7.6" x 10" 200GSM marker pad paper.

Digital brushes

Currently, I use an untextured version of the default Studio Pen available in Procreate on various opacity percentages for nearly everything, but I sometimes like to use these Copic-inspired brushes as well. I've also created a brush based on this Instagram tutorial for shading with rim lighting.
