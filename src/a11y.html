Accessibility Statement

  • All pages on this website are responsive for mobile users.
  • Includes persistent light-dark mode toggles.
  • US English set as the language for all pages.
  • Text and backgrounds conform to WCAG 2.2 at Level AA at the bare minimum
  • Font sizes set with EM/REM instead of PX.
  • Text is aligned with right, center, or left. Justify is never used.
  • Non-decorative images have alt text.
  • Pages include a 'skip to main content' link for keyboard users.
  • Pages include a 'back to top' button.
  • Javascript usage is kept to a minimum, but is required for some accessibility features.
  • Animated gifs are paused by default and can be played by hovering over them.
  • The site can be navigated at 200% zoom-in and at various screen resolutions.
  • Autoplay is never used.
